1. Download **AYA Space**: https://ayaneo.com/support/download
2. Install It
3. Download the language file from this GitHub
4. Copy file to
```
C:\Program Files (x86)\AYASpace\locale\
```
5. Restart of AYA Space
6. And choose new language in settings ;)
